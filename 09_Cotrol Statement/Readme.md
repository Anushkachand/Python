## Control Statements

| Type         | Statements            | Purpose               |
| ------------ | --------------------- | --------------------- |
| Conditional  | if, elif, else        | Decision making       |
| Looping      | for, while            | Repetition            |
| Control Flow | break, continue, pass | Change execution flow |

1. Conditional Statements (Decision Making)

```py
if condition:
    # code

elif condition:
    # code

else:
    # code
```

2.  Looping Statement

```py
# for loop
for i in range(n):
    # code

# while loop
while condition:
    # code
```
3. Control Flow Statements
```py
for i in range(5):
    if i == 3:
        break      # stops loop

    if i == 2:
        continue   # skip iteration

    pass           # do nothing
```
