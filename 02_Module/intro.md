# Python Modules

## What is a Module?
A **module** is a file containing Python code (functions, variables, or classes) that can be reused in other Python programs.

Python provides many **built-in modules** such as:
- csv
- datetime
- json
- math
- random
- sqlite3
- statistics
- tkinter
- turtle

Example: importing math module:
```
import math
 
print("Sin(0) =", math.sin(0))
print("Sin(30) =", math.sin(math.pi/6))
print("Sin(90) =", math.sin(math.pi/2))
```

Output:
```
Sin(0) = 0.0
Sin(30) = 0.49999999999999994
Sin(90) = 1.0
```
