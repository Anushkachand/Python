# PyJokes Module
- **pyjokes** is a fun Python module used to generate programming jokes. It’s mostly used for entertainment, demos, or learning Python modules.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Install Module (Pyjokes)

-  pip install Module name 
 eg: 
 ```
 pip install pyjokes
 ```

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

### 2. Exmple:
```
    import pyjokes
    
    joke = pyjokes.get_joke()
    print(joke)
```
👉 Output example:
```
    Why do programmers prefer dark mode? Because light attracts bugs!
```
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
▶️ Run Program
```
python joke.py
```

4. Get Multiple Jokes
```
import pyjokes

jokes = pyjokes.get_jokes()

    print(j)
```

## How to Handle import error?
- Handle import error
```
try:
    import pyjokes
except ImportError:
    print("Install module")
```
