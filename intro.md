
# 🐍 Python Comments 

## 1. What Are Comments?

- A **comment** is a part of the code that **Python ignores during execution**.
- A comment is a part of the coding file that the programmer does not want to execute

### Programmers use comments to:
- Explain code
- Make programs easier to understand
- Temporarily disable code during testing
- Add documentation

Comments improve **readability and maintainability** of programs.

---

# 2. Single-Line Comments

Single-line comments start with the **`#` symbol**.

### Syntax

```python
# This is a comment

Example 1 :
# This is a single-line comment
```
    print("This is a print statement.")
```
